# MATH πRATES ⚡

A premium, high-fidelity web application for **MATH πRATES** — an elite Maths coaching center focused on bringing conceptual clarity to students from Class 6-12, JEE, and NEET.

![Dashboard Preview](https://via.placeholder.com/1200x600?text=MATH+PIRATES+PREVIEW)

## ✨ Features

- **Premium Esthetics**: Modern, light-themed design with a "Frosted Glass" (Glassmorphism) UI.
- **Dynamic Backgrounds**: Immersive floating mathematical symbols background that reacts to page navigation.
- **High-Performance Animations**: Smooth, 60FPS scroll-triggered reveal animations.
- **Custom UI Components**: Hand-crafted custom dropdowns, interactive course cards, and animated testimonial sliders.
- **Direct Conversion**: Integrated enrollment flow that connects students directly via WhatsApp.
- **Fully Responsive**: Optimized for every device, from mobile phones to high-resolution desktops.

## 🚀 Tech Stack

- **Frontend**: React (Vite)
- **Styling**: Vanilla CSS (CSS3 Variable-based Design System)
- **Icons**: Custom SVG icons
- **Animations**: CSS transitions & keyframes

## 🛠️ Getting Started

### Prerequisites

- Node.js (v16+)
- npm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/sankhadipmondal05/MATH-PIRATES.git
   ```

2. Navigate to the project directory:
   ```bash
   cd MATH-PIRATES
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```
